# RT-DETR-Colaboratory-Training-Sample

[RT-DETR(v2 ※PyTorch版)](https://github.com/lyuwenyu/RT-DETR/tree/main/rtdetrv2_pytorch)をGoogle Colaboratory上で訓練しONNXのファイルをエクスポートするサンプルです。<br>
